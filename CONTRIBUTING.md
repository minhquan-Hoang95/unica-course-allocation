# 贡献指南

感谢对本项目的关注！

## 如何贡献

### 1. Fork 仓库
点击右上角 Fork 按钮

### 2. 创建分支
```bash
git checkout -b feature/your-feature-name
```

### 3. 提交更改
```bash
git add .
git commit -m "Add your message"
```

### 4. 推送到 fork
```bash
git push origin feature/your-feature-name
```

### 5. 创建 Pull Request
在原仓库点击 "New Pull Request"

## 开发规范

### 代码风格
- Python: 遵循 PEP 8
- 文档: 使用 Markdown

### 提交信息
格式：`[类型] 简短描述`
例如：
- `[feat] 添加数据处理脚本`
- `[fix] 修复 OPL 模型错误`
- `[docs] 更新 README`

### Issue 模板
使用预定义的 Issue 模板提交问题
