# Course Allocation Project

## 项目概述
课程分配系统 - 优化排课和资源分配

## 项目结构

```
unica-course-allocation/
├── docs/           # 文档、笔记、LaTeX 文件
├── opl/            # OPL 模型 (.mod, .dat 文件)
├── data/           # 原始和处理后的数据 (CSV, JSON)
├── scripts/        # Python 分析、可视化、数据处理脚本
├── results/        # 输出结果、图表、日志
├── tests/          # 单元测试或示例运行
├── .github/        # GitHub 配置 (issue 模板、workflows)
├── notebooks/      # Jupyter 笔记本（分析和原型）
├── examples/       # 示例数据集或模型运行
└── references/     # 论文、文档或规范
```
